# Tech Blog에 오신 것을 환영합니다!

이 블로그는 Next.js 16과 MDX를 사용하여 만들어진 기술 블로그입니다.

## 주요 기능

이 블로그는 다음과 같은 기능들을 제공합니다:

- **다크모드**: 라이트/다크 테마를 자유롭게 전환할 수 있습니다
- **태그 시스템**: 포스트를 태그별로 분류하고 검색할 수 있습니다
- **MDX 지원**: 마크다운에서 React 컴포넌트를 사용할 수 있습니다
- **반응형 디자인**: 모든 디바이스에서 최적화된 경험을 제공합니다

## 기술 스택

```typescript
const techStack = {
  framework: 'Next.js 16',
  language: 'TypeScript',
  styling: 'Tailwind CSS',
  content: 'MDX',
};
```

## 마크다운 예시

일반적인 마크다운 문법을 모두 지원합니다:

1. 순서 있는 목록
2. **굵은 글씨**와 *기울임*
3. [링크](https://nextjs.org)도 사용할 수 있습니다

> 인용구도 깔끔하게 표시됩니다.

## 앞으로의 계획

앞으로 이 블로그에서는:

- 웹 개발 튜토리얼
- 프로그래밍 팁과 트릭
- 기술 트렌드 분석
- 개인 프로젝트 소개

등 다양한 콘텐츠를 공유할 예정입니다.
