## 무슨 일이 있었나요?

OpenAI의 코드 어시스턴트 **Codex**의 모델 컨텍스트 크기가 슬그머니 줄어들었습니다. 372k(약 37만 2천 토큰)에서 272k(약 27만 2천 토큰)로, 무려 10만 토큰 가까이 축소된 것인데요. 이 변경은 GitHub PR #33972를 통해 `release/0.144` 브랜치에 백포트(backport)되면서 세상에 알려졌습니다.

> **백포트(backport)란?** 최신 버전에서 만든 변경 사항을 이전 릴리스 버전에도 반영하는 작업을 말해요. 즉, 이번 컨텍스트 축소는 원래 다른 곳에서 이미 적용됐던 변경이 0.144 릴리스에도 옮겨진 것으로 보입니다.

- **PR 번호**: #33972

- **대상 브랜치**: `openai:release/0.144`
- **커밋**: 1개 (`b06f4fa`, 커밋 메시지 "Backport refreshed bundled model metadata")
- **변경 파일**: JSON 1개

- **병합일**: 2026년 7월 18일
- **검사(CI)**: 36개 체크 통과

특이한 점은 이 변경이 **별도의 논의나 리뷰 코멘트 없이** 조용히 병합됐다는 것입니다. 보통 모델 스펙처럼 중요한 값이 바뀌면 활발한 토론이 오갈 법도 한데, 이번 건은 단일 커밋으로 깔끔하게 처리됐습니다.
